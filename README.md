# GRC Evidence Automation

Scripts to automate compliance evidence collection and continuous control monitoring for ISO 27001, SOC 2, HIPAA, and ISO 22301.

## What it does
- Pulls evidence from cloud/SaaS sources on a schedule
- Maps evidence to control requirements
- Flags stale or missing evidence before audit

## Stack
Python, REST APIs (Sprinto-style), scheduled jobs

## Status
Sanitized examples from production GRC automation work. Not connected to live systems.
